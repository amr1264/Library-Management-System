📚 Library Management System

Console Application / OOP Project

Welcome to the official presentation page of the Library Management System. This project is a solid implementation of Object-Oriented Programming (OOP) architectures in C#, completely decoupling business logic from the presentation layer.

✨ Key Technical Features

  • Robust Validation Framework: Powered by int.TryParse to completely immunize the runtime against malicious or broken formatting crashes.
  • Fluent Search Index: Built-in partial word matching using .Contains() and string normalization (.ToLower().Trim()).
  • Advanced OOP Design: Strict application of Abstraction, Composition patterns, and deep encapsulation using dedicated Service Layers.

🚧 Learning Journey & Evolution

This system represents a vital milestone in my journey toward professional .NET Backend Engineering. It was built step-by-step to understand the real-world execution behavior of memory lifecycles, static states, and reference exception troubleshooting.
